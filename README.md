# IS-LM Interactive Macroeconomics Simulator 📈

An interactive IS-LM macroeconomic simulator built with Python and Streamlit. This project aims to bridge classic economic theory with modern, smooth UI visualization, helping you learn macroeconomics better through interactive charts.

## 🚀 How to Run Locally

You can easily get this simulator up and running on your computer in just two steps:

### 1. Install Dependencies
This project requires `streamlit` for the web interface, along with `plotly` and `numpy` for data calculation and charting. Open your terminal and run the following command to install them:

```bash
pip install streamlit plotly numpy
```
### 2. Launch the Application
Once the dependencies are installed, start the simulator by running:

```bash
streamlit run app.py
```
💡 Tip: After running the command above, your default web browser will automatically open a new tab (usually at http://localhost:8501), and you can start interacting with the model right away!
